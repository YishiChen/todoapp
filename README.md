# Todoapp
A todoapp built using Postgresql, Express.js, React.js and Node.js. The repository includes code for both the back- and front-end.

# Instructions
To test out the application, you need to install and run a postgresql server with the following credentials:

const pool = new Pool({
  user: "postgres",
  password: "",
  host: "localhost",
  port: 5432,
  database: "todoapp"
});
